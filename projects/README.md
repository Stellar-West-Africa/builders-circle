# Project Submission Guide

Add your project to the Builders Circle showcase by following the steps below:

1. **Create a new file** in this directory:
   ```
   projects/your-project-name.md
   ```

2. **Copy the template** below and fill in your details

3. **Submit a PR** to this repository with your project file

## Project Template

Replace the placeholder values with your own:

```markdown
# Project Name

**Builder(s)**: List your team members

**Status**: Planning | In Development | Beta | Live

**Description**: A short description of what your project does and the problem it solves.

## Repository

**GitHub**: [github.com/your-username/your-repo](https://github.com/your-username/your-repo)
**Demo**: *Link or "Coming soon"*
**Documentation**: *Link or "Coming soon"*

## Milestones

- [x] Completed milestone
- [ ] Upcoming milestone

## Looking For (Optional)

- **Role** — Short description of the role you're looking for

## Contact

- GitHub: @handle-of-the project's-primary-contact
- X: @handle-of-the-project's-primary-contact
```
